map 方法总结

主要作用

  • 对数组进行二次操作，并返回一个数组

语法

var new_array = arr.map(function callback(currentValue[, index[, array]])
{ // Return element for new_array }[, thisArg])

参数

  1. callback 回调函数
  • 生成新的数组
  1. currentValue 当前处理元素
  • index 当前处理元素的索引值
  • 调用 map 的元素组

原理讲解

  • map 不会改变元素组 所以你需要一个新的数组来接受返回的数组
  • map 函数会对有索引值的元素进行遍历
